## ✨ Was macht die App?

Der **YouTube Channel Watcher** prüft beim Windows-Start automatisch alle gespeicherten Kanäle auf neue Videos – ganz ohne YouTube-Konto, ohne API-Key und ohne Anmeldung.

- 🔔 **Automatisch beim Start** – läuft still im Hintergrund
- 📺 **Neue Videos sofort sehen** – Fenster öffnet sich nur wenn wirklich etwas Neues da ist
- ▶️ **Ein Klick** – direkt zu YouTube
- 🔒 **Keine Anmeldung** – nutzt öffentliche RSS-Feeds
- 💾 **Keine Cloud** – alle Daten bleiben lokal auf deinem PC

## 📥 Download & Installation

1. [**Installer herunterladen**](https://github.com/Rabenstaub/yt-channel-watcher/releases/latest)
2. `YTChannelWatcher_Setup_v1.1.0.exe` ausführen
3. Installationsanleitung folgen
4. Fertig – die App startet automatisch mit Windows

> **Hinweis:** Da die App nicht von Microsoft signiert ist, kann Windows SmartScreen beim ersten Start eine Warnung anzeigen. Einfach auf „Weitere Informationen" → „Trotzdem ausführen" klicken.

## 🚀 Erste Schritte

1. App starten → Tab **„Kanäle verwalten"** öffnet sich automatisch
2. YouTube-Handle eingeben, z. B. `@MythenMetzger`
3. Auf **＋ Hinzufügen** klicken
4. Auf **🔄 Jetzt prüfen** klicken – neue Videos erscheinen sofort

Beim nächsten Windows-Start prüft die App automatisch alle Kanäle.
